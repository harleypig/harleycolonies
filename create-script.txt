# Create Python Script

Create a python script called gen-modlist to read toml files in this
directory.

## Features

- Ignore pack.toml and index.toml.
- Create a markdown file following the template below.
- The url to use for <ProjectUrl> is `metadata.curseforge.website`.
- <Class> is found in <ProjectUrl>
  - https://www.curseforge.com/minecraft/<Class>/mod"
- <Category> is `metadata.curseforge.categories`.
  - If multiple categories exist, include the mod in each class or mod instance
- Sort the output alphabetically by <Class>, then <Category>, then <ModName>.

## Template

    # ModList for HarleyColonies

    ## <Class>

    ### <Category>

    - [<ModName>](<ProjectUrl>)
