---
title: Resource Pack Overrides
---

**Resource Pack Overrides** is a library mod that provides functionality
for overriding resource pack behavior. The mod allows modpack creators to
configure resource pack loading and override behavior through configuration
files.

### Configuration

Below are the available options and their defaults. Paths and values use
the same names as in the mod's config file, `resourcepackoverrides.json`.

#### General Settings

- `schema_version` (default: `"2"`)
  - Configuration schema version. Used internally to track config format
    changes. This should generally not be modified manually.

### Modpacks

This mod is installed in the following modpacks:

- harleycolonies-1.21.1
