---
title: Stylecolonies
---

**Stylecolonies** adds style-related features and enhancements for
MineColonies. The mod provides additional style options and customization
for MineColonies colonies, enhancing the building and customization
experience.

### Features

- **Style Enhancements**: Additional style options for MineColonies.
- **Customization**: More customization options for colonies.
- **MineColonies Integration**: Seamlessly integrates with MineColonies.

### Modpacks

This mod is installed in the following modpacks:

- harleycolonies-1.21.1
