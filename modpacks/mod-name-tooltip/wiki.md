---
title: Mod Name Tooltip
---

**Mod Name Tooltip** displays which mod an item or block comes from in
tooltips. The mod helps players identify modded content by showing the
source mod name, making it easier to understand which mods add which
content.

### Features

- **Mod Identification**: Shows which mod items and blocks come from.
- **Tooltip Display**: Displays mod name in item tooltips.
- **Content Organization**: Helps organize and identify modded content.

### Modpacks

This mod is installed in the following modpacks:

- harleycolonies-1.21.1
