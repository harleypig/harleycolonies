# Server Operational Documentation

This directory contains operational documentation for the HarleyColonies
modpack server. These documents contain sensitive information including
server setup procedures, security configurations, and infrastructure
details. **Do not make these documents publicly accessible.**

## Documentation Index

- **[Setup](setup.md)** - Initial server setup and configuration
  procedures
- **[Deployment](deployment.md)** - Server updates, maintenance, and
  lifecycle management
- **[Security](security.md)** - Security configuration, firewall rules,
  and access controls
- **[Infrastructure](infrastructure.md)** - Hosting provider setup,
  DNS configuration, and networking

## Quick Reference

### First Time Setup
1. Read [Setup](setup.md) for initial server configuration
2. Configure security per [Security](security.md)
3. Set up infrastructure per [Infrastructure](infrastructure.md)

### Regular Operations
- Updates: See [Deployment](deployment.md)
- Security maintenance: See [Security](security.md)
- Infrastructure changes: See [Infrastructure](infrastructure.md)

### Emergency Procedures
- Server issues: See [Deployment](deployment.md#emergency-procedures)
- Security incidents: See [Security](security.md#incident-response)
- Infrastructure problems: See [Infrastructure](infrastructure.md#disaster-recovery)
