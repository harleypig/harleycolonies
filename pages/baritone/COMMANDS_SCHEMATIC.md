# Schematic Tools

This document contains all commands related to schematic tools.

## Commands

### `litematica`

**Description**: Builds the loaded schematic  

Build a schematic currently open in Litematica.

Usage:
> litematica
> litematica <#>

---

### `schematica`

**Description**: Builds the loaded schematic  

Builds the schematic currently open in Schematica.

Usage:
> schematica
