Client/Server: both

**ModernFix** is a performance optimization mod that fixes various issues
and improves performance in Minecraft. The mod includes fixes for memory
leaks, performance optimizations, and compatibility improvements. It
provides configurable options to control which fixes are applied, allowing
server administrators to fine-tune performance based on their specific
needs.

### Configuration

Below are the available options and their defaults. Paths and values use
the same names as in the mod's config file, `modernfix-common.toml`.

#### JEI Plugin Settings

- `blacklist_async_jei_plugins` (default: `["jepb:jei_plugin"]`)
  - These JEI plugins will be loaded on the main thread. Some JEI plugins
    may need to be loaded synchronously to prevent issues. Add plugin IDs
    to this list to force them to load on the main thread.
