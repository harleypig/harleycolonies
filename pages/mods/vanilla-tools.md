Client/Server: both

**Vanilla Plus Tools** adds additional tool variants and improvements to
vanilla tools. The mod provides more tool options while maintaining a
vanilla-like feel, enhancing the tool progression system.

### Features

- **Additional Tools**: More tool variants and options.
- **Vanilla Feel**: Maintains vanilla Minecraft aesthetics.
- **Tool Improvements**: Enhanced tool functionality.
