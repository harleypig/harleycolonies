Client/Server: both

**FullGrownCropMarker** adds visual markers to fully grown crops, making
it easier to identify which crops are ready to harvest. The mod provides
visual indicators for crop maturity, improving farming efficiency.

### Features

- **Crop Markers**: Visual markers for fully grown crops.
- **Harvest Indicators**: Easy identification of ready-to-harvest crops.
- **Farming Aid**: Makes farming more efficient and convenient.
