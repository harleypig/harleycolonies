Client/Server: both

**Searchables** adds search functionality to various in-game interfaces,
making it easier to find items, recipes, and other content. The mod
provides enhanced search capabilities across multiple interfaces.

### Features

- **Enhanced Search**: Improved search functionality in various interfaces.
- **Multi-Interface Support**: Works across multiple game interfaces.
- **Better Organization**: Makes finding content easier and faster.
