---
title: Smithing Template Viewer for JEI/EMI
---



Client/Server: client

**Smithing Template Viewer** adds smithing template information to JEI
(Just Enough Items) and EMI (EMI Item Index), allowing players to view
which trim materials are available for smithing templates. The mod
displays trim materials in the recipe viewer, making it easier to see what
materials can be used with each template.

### Configuration

Below are the available options and their defaults. Paths and values use
the same names as in the mod's config file, `smithingtemplateviewer-common.toml`.

#### Display Settings

- `display_materials` (default: `true`)
  - Whether trim materials should be registered into EMI/JEI. When
    enabled, trim materials appear in the recipe viewer for smithing
    templates.
