---
title: JEI Integration: Reborn
---



Client/Server: both

**JEI Integration: Reborn** adds various tooltip enhancements to JEI
(Just Enough Items), displaying additional information about items when
viewing recipes. The mod can show burn time, durability, enchantability,
food values, registry names, max stack size, tags, translation keys, and
data components. All tooltips are configurable and can be set to always
show, show on shift, show in debug mode, or be disabled.

### Configuration

Below are the available options and their defaults. Paths and values use
the same names as in the mod's config file, `jei_integration_reborn-client.toml`.

#### Tooltip Options

All tooltip options support the following modes: `DISABLE` (never show),
`ENABLE` (always show), `SHIFT` (show when holding shift), `DEBUG` (show
in debug mode), `SHIFT_AND_DEBUG` (show when holding shift or in debug
mode).

- `Tooltip_Options.burnTimeTooltip` (default: `"DISABLE"`)
  - Display burn time tooltip for fuel items.

- `Tooltip_Options.durabilityTooltipMode` (default: `"DISABLE"`)
  - Display durability tooltip for tools and armor.

- `Tooltip_Options.enchantabilityTooltipMode` (default: `"DISABLE"`)
  - Display enchantability tooltip for enchantable items.

- `Tooltip_Options.foodTooltipMode` (default: `"DISABLE"`)
  - Display food value tooltip for food items.

- `Tooltip_Options.registryNameMode` (default: `"DISABLE"`)
  - Display registry name (item ID) tooltip.

- `Tooltip_Options.maxStackSizeTooltipMode` (default: `"DISABLE"`)
  - Display maximum stack size tooltip.

- `Tooltip_Options.tagsTooltipMode` (default: `"DISABLE"`)
  - Display item tags tooltip.

- `Tooltip_Options.translationKeyTooltipMode` (default: `"DISABLE"`)
  - Display translation key tooltip.

- `Tooltip_Options.dataComponentsMode` (default: `"DISABLE"`)
  - Display data components tooltip.
