Client/Server: both

**BetterF3** enhances the F3 debug screen with better formatting, more
information, and customizable display options. The mod makes the debug
screen more readable and useful, especially in modded environments where
more information is often needed.

### Features

- **Enhanced Debug Screen**: Better formatted F3 debug screen.
- **More Information**: Additional useful debug information.
- **Customizable**: Configurable display options and formatting.
